# Сайт «АкваЛайн» — ремонт водопроводов без отключения воды

Готовый статичный сайт (HTML/CSS/JS). Работает без сервера — можно открыть двойным кликом.

## Как посмотреть сайт
Откройте файл **index.html** двойным кликом — он запустится в браузере. Все страницы связаны между собой через меню.

## Страницы
| Файл | Страница |
|------|----------|
| `index.html` | Главная |
| `uslugi.html` | Услуги |
| `o-kompanii.html` | О компании |
| `proekty.html` | Объекты |
| `kontakty.html` | Контакты |

## Что нужно заменить на реальные данные (сейчас стоят «заглушки»)
Просто скажите мне новые значения — я поменяю их сразу на всех страницах:

- **Название компании** — сейчас «АкваЛайн» (можно оставить или заменить).
- **Телефоны** — сейчас `+7 (495) 000-00-00` и `+7 (812) 000-00-00`.
- **E-mail** — сейчас `info@aqualine.ru`.
- **Логотип** — сейчас нарисованная иконка-капля. Можно заменить на ваш логотип.
- **Объекты/проекты** — сейчас типовые примеры. Пришлите фото и описания реальных работ — добавлю.
- **Карта** на странице «Контакты» — добавлю Яндекс.Карту, когда будет адрес/зона работ.
- **Политика конфиденциальности** — сделаю отдельную страницу, когда будет текст.

## Форма заявки
Сейчас форма показывает сообщение «Заявка отправлена», но письма пока никуда не уходят —
для реальной отправки нужен хостinг/сервис форм. Настрою, когда появится хостинг
(можно подключить отправку на e-mail или в Telegram).

## Что понадобится для запуска в интернете
1. **Домен** (например, `aqualine.ru`) — регистрируется у регистратора.
2. **Хостинг** — подойдёт любой простой хостинг для статичных сайтов.
3. Я помогу с настройкой отправки формы и подключением карты/аналитики.

## Технические заметки
- Стили — в `css/styles.css`, скрипты — в `js/main.js`.
- Папка `.claude/` нужна только для локального предпросмотра, на хостинг её загружать не обязательно.
